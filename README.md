# Hospital Management System

Hospital Management System is a desktop-based application built with C# Windows Forms and .NET 8. The system is designed to manage essential hospital operations efficiently through dedicated modules for patients, doctors, nurses, admins, appointments, prescriptions, medical records, and reporting.

## Features

- Secure user login and registration system
- Separate roles for patients, doctors, nurses, and administrators
- Patient profile and information management
- Appointment scheduling and cancellation functionality
- Doctor prescription handling and management
- Medical history management with add, edit, and delete options
- Nurse support and assistance management
- Hospital income and appointment report generation
- Integration with SQL Server database

## Technologies Used

- C#
- .NET 8
- Windows Forms
- SQL Server
- ADO.NET

## Project Purpose

The primary objective of this project is to develop a hospital management desktop application while gaining practical experience in database integration, CRUD operations, role-based access management, and Windows Forms development using C# and SQL Server.
